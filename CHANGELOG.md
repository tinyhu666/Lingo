# 更新日志

本文件记录客户端面向用户的重要变更，按版本倒序维护。

写法约定：

1. 面向普通用户表达，优先说明体验变化与业务价值，避免内部接口、SDK、部署、统计链路、服务迁移等实现细节。
2. 只有在确实无法提炼为用户价值时，才使用“修复已知问题”兜底；兜底内容直接写“修复已知问题”，不要再重复添加“修复：”前缀。
3. 每个版本统一使用“更新日志”作为小标题，正文按“1. 新增：… / 2. 优化：… / 3. 修复：…”的顺序展开。
4. 同一条文案尽量只表达一个重点，便于同步到 GitHub Release、官网与客户端内展示。

## [0.6.7] - 2026-04-03

### 更新日志

1. 优化：队友翻译快捷键默认改为 Alt+E，设置页里会始终显示并可直接修改。
2. 修复：队友翻译在游戏内现在会优先直接识别当前聊天区域并显示悬浮译文，拿不到当前游戏窗口时才退回框选，避免按下热键后切出游戏或在副屏上看不到选择层。
