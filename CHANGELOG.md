# 更新日志

本文件记录客户端面向用户的重要变更，按版本倒序维护。

## [Unreleased]

### 更新日志

1. 暂无更新。

## [0.3.6] - 2026-03-12

### 更新日志

1. 新增客户端顶部界面语言切换器，支持简体中文、English、Русский，默认简体中文并支持本地持久化。
2. 客户端可见文案全面接入 i18n，覆盖导航、首页、翻译风格、常用语、关于、设置、更新提示与标题栏提示。
3. 修复 Windows 客户端透明窗口四角残留问题，统一外层裁切与圆角策略，窗口拖动与恢复后表现稳定。
4. 统一标题栏最小化与关闭按钮的视觉尺寸和点击区域，保持一致交互反馈，关闭按钮保留危险态悬停色。
## [0.3.5] - 2026-03-11

### 更新日志

1. 优化客户端整体背景层次，降低过度透明造成的可读性问题，改为更深的渐变色表现。
2. 放大左上角品牌区 Logo 与文字，增强品牌识别度与视觉权重。
3. 移除品牌区边框与背景底板，保留简洁品牌信息展示。

## [0.3.4] - 2026-03-11

### 更新日志

1. 修复 macOS 客户端窗口四个角出现白色直角的问题，补齐透明窗口生效配置。
2. 修复应用图标外圈白边问题，移除图标源文件外层浅色描边。
3. 重新生成并同步全平台图标资源（包含 `icon.icns`、`icon.ico`、前端展示图标）。

## [0.3.3] - 2026-03-11

### 更新日志

1. 修复首页与常用语页超出屏幕区域无法滚动的问题。
2. 修复左侧导航首项与右侧首模块顶部未对齐的问题。
3. 修复翻译风格、常用语、关于页面切换时首模块出现纵向偏移的问题。
4. 修复“当前风格”“常用语库”标签冗余展示问题。
5. 将常用语容量计数（`x/20`）移动到标题右侧，操作区仅保留“新增常用语”和“保存”按钮。
6. 窗口启用透明背景配合圆角裁切，修复客户端四角直角残留问题。
7. 统一首页、翻译风格、常用语页面标题区排版与对齐间距，优化阅读体验。
8. 统一工作区滚动策略为纵向优先，减少横向滚动干扰。
9. 更新日志样式统一为“更新日志 + 数字编号”格式。

## [0.3.2] - 2026-03-11

### 更新日志

1. 修复 GitHub Release 自动日志与手写内容混合导致的重复问题。
2. 修复发布文案中“版本升级到 x.y.z”这类无效条目残留问题。
3. 修复首页与关于页信息类图标风格不一致问题，统一为 `CircleInfo`。
4. 规范发布日志来源，改为从 `CHANGELOG.md` 按版本提取，发布文案结构统一、可维护。
5. 优化更新日志文案，避免冗余描述和重复信息。

## [0.3.1] - 2026-03-11

### 更新日志

1. 修复“关于”导航图标语义不匹配问题，替换为信息类图标。
2. 修复应用内与系统图标显示比例异常，提升 Dock 与任务栏清晰度。
3. 修复更新说明显示策略，仅在检测到新版本时展示，避免重复历史日志干扰。
4. 统一客户端圆角体系，修复外边框直角暴露与模块圆角不一致问题。
5. 顶栏去除多层边框与背景叠加，窗口操作区视觉更简洁。
6. 侧栏移除重复品牌区与“导航”字样，整体对齐与层级更清晰。
7. 统一“翻译风格”命名，移除“模式”后缀，文案表达一致。
