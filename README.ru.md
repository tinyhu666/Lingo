# Lingo | AI-перевод игрового чата

[简体中文](./README.md) | [English](./README.en.md) | Русский

[Сайт](https://lingo.ink) | [Последний релиз](https://github.com/tinyhu666/Lingo/releases/latest) | Windows / macOS | MIT

Lingo — это AI-клиент перевода игрового чата для Windows и macOS, созданный для командного общения с минимумом лишних переключений.

Это не универсальный переводчик. Это hotkey-first workflow для командного общения в игровом чате: по хоткею Lingo копирует текущую реплику, отправляет её на серверный AI для перевода или полировки и вставляет результат обратно в поле ввода. Вы остаетесь в матче, а Lingo снижает количество лишних переключений и пауз вокруг каждого сообщения.

## Почему это удобно в матче

- `Меньше переключений`: hotkey-first поток от копирования до возврата текста
- `Фразы, которые можно сразу отправлять`: упор на короткий игровой чат с сохранением терминов по скиллам, предметам и коллам
- `Контекст игры`: поддерживаются Dota 2, League of Legends, World of Warcraft, Overwatch и общий режим
- `Server-side AI`: маршрутизация моделей управляется на бэкенде, без логина и ручной настройки model API на клиенте
- `Удобно на десктопе`: Windows / macOS, обновления внутри приложения и интерфейс на китайском, английском и русском

## Основные возможности

- Глобальный хоткей для перевода, по умолчанию: macOS `⌘+T`, Windows `Alt+T`
- Авто-копирование -> перевод / rewrite -> авто-вставка обратно в текущее поле ввода
- Стили `Normal / Pro / Toxic` для разного тона и темпа общения
- Быстрые фразы для частых реплик
- Многоязычный перевод, интерфейс клиента на упрощенном китайском, английском и русском

## Как это работает

1. Выделите или введите текст в игровом чате.
2. Нажмите горячую клавишу перевода, и Lingo обратится к серверному переводу, чтобы превратить мысль в понятную для союзников реплику.
3. Переведенная строка автоматически вернется в текущее поле ввода, чтобы вы могли отправить её почти не выходя из игры.

Для повторяющихся реплик можно сразу использовать сохраненные быстрые фразы.

## Требования к окружению

- Node.js 18+
- Rust 1.77+
- Зависимости сборки Tauri

### macOS

- Xcode Command Line Tools
- При первом запуске выдайте разрешения Accessibility/Automation в System Settings

### Windows

- Visual Studio C++ Build Tools (Desktop development with C++)
- WebView2 Runtime

## Сборка и релиз

```bash
npm run tauri build
```
