# Lingo - Ваш голос в игре

[简体中文](./README.md) | [English](./README.en.md) | Русский

Lingo — это клиент перевода внутриигрового чата на базе AI-моделей, но по ощущениям он скорее ваш голос в игре.

Вы думаете о том, что хотите сказать, а Lingo помогает сказать это быстрее, понятнее и на языке, который поймут союзники, а затем сразу возвращает текст в чат. По одной глобальной горячей клавише приложение проходит весь цикл: копирование, перевод и вставка обратно.

- Превращает вашу мысль в сообщение, которое понимают тиммейты
- Поддержка глобальных горячих клавиш: авто-копирование -> перевод -> авто-вставка
- Поддерживаются быстрые фразы и многоязычный перевод
- Перевод предоставляется централизованным сервером; не требуется вход или настройка параметров model API на клиенте
- Поддержка Windows / macOS
- Язык интерфейса клиента: упрощенный китайский, английский и русский

## Почему Lingo — это ваш голос в игре

- Не нужно сворачивать игру и открывать отдельный переводчик
- Не нужно снова и снова вручную копировать, вставлять и править фразы
- Не нужно на ходу формулировать мысль на другом языке в напряженный момент
- Сообщение быстрее попадает в чат, а игровой ритм не ломается

## Как это работает

1. Выделите или введите текст в игровом чате.
2. Нажмите горячую клавишу перевода (по умолчанию: macOS `⌘+T`, Windows `Alt+T`).
3. Lingo считывает содержимое буфера обмена, переводит текст и вставляет результат обратно в текущее поле ввода.
4. Для часто используемых фраз можно отправлять сохраненные быстрые шаблоны напрямую.

## Требования к окружению

- Node.js 18+
- Rust 1.77+
- Зависимости сборки Tauri

### macOS

- Xcode Command Line Tools
- При первом запуске выдайте разрешения Accessibility/Automation в System Settings

### Windows

- Visual Studio C++ Build Tools (Desktop development with C++)
- WebView2 Runtime

## Сборка и релиз

```bash
npm run tauri build
```
