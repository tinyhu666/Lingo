# Lingo - Клиент перевода внутриигрового чата

[简体中文](./README.md) | [English](./README.en.md) | Русский

Lingo — это клиент перевода внутриигрового чата на базе AI-моделей для общения в реальном времени в разных игровых сценариях. Приложение запускает перевод по глобальной горячей клавише и автоматически выполняет копирование, перевод и вставку обратно.

- Поддержка Windows / macOS
- Язык интерфейса клиента: упрощенный китайский, английский и русский
- Поддержка глобальных горячих клавиш: авто-копирование -> перевод -> авто-вставка
- Перевод предоставляется централизованным сервером; не требуется вход или настройка параметров model API на клиенте
- Поддерживаются быстрые фразы и многоязычный перевод

## Как это работает

1. Выделите или введите текст в игровом чате.
2. Нажмите горячую клавишу перевода (по умолчанию: macOS `⌘+T`, Windows `Alt+T`).
3. Lingo считывает содержимое буфера обмена и автоматически вставляет перевод обратно.

## Требования к окружению

- Node.js 18+
- Rust 1.77+
- Зависимости сборки Tauri

### macOS

- Xcode Command Line Tools
- При первом запуске выдайте разрешения Accessibility/Automation в System Settings

### Windows

- Visual Studio C++ Build Tools (Desktop development with C++)
- WebView2 Runtime

## Сборка и релиз

```bash
npm run tauri build
```
